Anonymous IP assignment using blind RSA signatures. Complete with academic research paper and formal security proofs.  
`Rust • Cryptography • Zero-Knowledge • Microservices`

**[EmotionalChain](https://github.com/ChronoCoders/EmotionalChain)** - Proof-of-Emotion Blockchain  
Novel consensus mechanism combining biometric data with traditional staking. zk-SNARK privacy layer.  
`Rust • P2P • Consensus • Privacy`

Military-grade encrypted file transfer. RSA-4096 + AES-256-GCM with self-destruct capabilities.  
`Rust • Cryptography • Security • Networking`

**[cargo-sane](https://github.com/ChronoCoders/cargo-sane)** - Rust Dependency Manager
